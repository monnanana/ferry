## 游戏概述
- 类型：赛博朋克文字冒险 + 资源管理
- 目标时长：15-20分钟
- 核心主题：人性、金钱、罪恶、背叛、恐惧交织
- 基调：惊悚紧张，结局有一点点希望

## 核心数值系统
- 体力(HP)：0-100，归零=途中死亡
- 时间：120min倒计时，归零=被追上
- 警戒等级：Lv.1🟢→Lv.4🔴，Lv.4=围捕
- 资金：购买补给+途中贿赂用
- 燃料状态：独立指示灯（充足/有备用/未检查⚠/安全）

## 5个角色

| 角色 | HP | 初始金 | 核心动机 | 独占能力 |
|------|-----|------|---------|---------|
| 零 Zero | 100 | ¥80 | 曾被装进箱子贩卖，赎罪 | 钻通风道/货柜躲避检查 |

## 补给系统
可购买（可买多份）：
- 食物补给 ¥20：休息站吃，体力+25
- 氧气包 ¥30：休息站用，低氧区安全
- 燃料备用罐 ¥25：休息站检查，防迫降
- 急救包 ¥35：体力+40，结局可救婴儿
- 镇静剂 ¥40：警戒-1
- 干扰器 ¥50：阻挡一次扫描

## 游戏流程
```
选角色 → 买补给（提示预留资金）→ 出发

飞行段1(出城之夜) → 检查站[道德:老人被踢] → 飞行段2(工业废区/儿歌)
→ 飞行段3(奇怪的重量/悬念) → 休息站1 → 低氧区
→ 可疑接触[道德:父亲在航道] → 休息站2
→ 飞行段4(引擎颤抖) → 燃料危机 → 飞行段5(幻觉/值得吗)
→ 疲劳场景 → 飞行段6(终点在望) → 巡逻遭遇
→ 婴儿哭泣(Scene11) → 委托人取消任务[道德:最大反转]
→ Gate-7 → 结局
```

## 道德分叉（6个）
1. **检查站老人**：帮/不帮/陈叔举报 → G.choices.helpedOldMan
2. **箱子的重量**：靠近听/调高音乐 → G.choices.heardBox
3. **父亲在航道**：告诉他活着/沉默/承诺/离开 → G.choices.toldFather
4. **幻觉"值得吗"**：4个纯心理选择，无道具可解 → G.choices.forChild
5. **休息站走私者**：帮/给补给/无视/观察
6. **委托人取消任务**：拒绝/装没收到/打开箱子再决定→看见孩子还选钱=背叛者

## 两个反转
- **反转1（中期）**：父亲在航道等着 → 委托人就是发追捕令的人
- **反转2（结局前）**：父亲跟着信号到了终点（如果告诉过他孩子活着）

## 结局体系（7种×5角色=35段文字）
| 结局 | 触发条件 | 基调 |
|------|---------|------|
| ✨父亲的奇迹 | 拒绝委托人+孩子活+告诉了父亲+评分≥9 | 温情 |
| 🌅摆渡完成 | 拒绝委托人+孩子活+评分≥7 | 阳光带代价 |
| 🌧代价 | 拒绝委托人+孩子活+评分≥4 | 温情沉重 |
| 😔没能撑住 | 拒绝委托人+孩子活+评分<4 | 勉强完成 |
| 🌑棋子 | 接受委托人/无视消息 | 黑暗，被出卖 |
| 🔪背叛者 | 打开箱子看见孩子，还是选了钱 | 最黑暗 |
| 💀/🚨 死亡/围捕 | 体力/时间/警戒归零 | 各角色专属 |
